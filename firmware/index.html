🚀 Instalación del Firmware

ℹ️ Información: Este firmware está optimizado para LILYGO T-Watch S3. Asegúrate de tener el dispositivo correcto antes de continuar.
⚠️ Advertencia: La instalación del firmware borrará todos los datos del dispositivo. Asegúrate de hacer un respaldo si es necesario.
Tu navegador no soporta Web Serial. Usa Chrome, Edge o un navegador compatible. No se detectó ningún puerto serie. Conecta tu dispositivo ESP32-S3 via USB.

📋 Instrucciones de Uso

  1. Conecta tu LILYGO T-Watch S3 al puerto USB de tu computadora
  2. Presiona el botón "📥 Instalar USB Army Knife" arriba
  3. Selecciona el puerto serie correspondiente a tu dispositivo
  4. Espera a que complete la instalación (puede tardar unos minutos)
  5. Una vez completado, desconecta y reconecta el dispositivo
  6. Accede a la interfaz web desde http://192.168.4.1 (modo AP)
